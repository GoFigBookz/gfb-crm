# Brand / Logo assets — Go Fig Bookz / Figgy

Central place for the brand files so they're never lost.

| File | What it is |
|------|------------|
| `go-fig-bookz-logo.jpg` | The firm logo (flowing calligraphy + swoosh, lime green). The brand reference. |
| `figs-mascot.png` | The **Figs mascot** as a high-res **transparent PNG** — drop onto the "i" of "Figgy" in Canva/Figma. |
| `figs-mascot.svg` | The Figs mascot as vector (scales infinitely). |

## Brand colors
- Lime green (primary / "Bookz" green): `#8BC53F`
- Mid green: `#6FAE2E`  ·  Deep olive: `#4D7C0F`
- App primary: lime-600 `#65A30D`

## Making the Figgy wordmark
Build it in Canva/Figma: type "Figgy" in a flowing script font (Pacifico / Dancing
Script / Great Vibes), drop `figs-mascot.png` on the i, color the text `#8BC53F`,
export a transparent PNG/SVG. Send the file to drop into the app header.
